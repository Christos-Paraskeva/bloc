## Installation
- Clone the repo
- `cd` to the project folder
- Run `npm install`
- Run `brew install mongodb`
- Create a local MongoDB called `bloc`

To test:
- Run `npm test`

To run the application on localhost:
- Run `node server.js`
- Visit http://localhost:8080/
